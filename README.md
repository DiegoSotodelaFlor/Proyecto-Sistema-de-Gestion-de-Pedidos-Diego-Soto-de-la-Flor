Ingenieria inversa:
Se debe ejecutar el archivo Main.java. de la carpeta src El programa recoje los datos de Productos Digitales y Físicos que heredan de Producto,
del Cliente y del Pedido y muestra un resumen de los atributos de cada clase.
También muestra un descuento y un porcentage de IVA y el total de euros que cuestan los productos.

En cada clase hay atributos, constructor, get y set de cada atributo y métodos poo funciones que se relacionan o no con otras clases.

La parte normal del proyecto es la carpeta generado y la src la mejorada.
